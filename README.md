# CareerCompass

CareerCompass is a project designed to help individuals navigate their career paths by providing personalized recommendations, insights, and tools to achieve their professional goals.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/CareerCompass.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CareerCompass
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Environment Variables

Before running the project, ensure you set up the required environment variables. Use the `.env.example` file as a reference:

1. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
2. Update the values in `.env` as needed

## Usage

1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to `http://localhost:5000`.
